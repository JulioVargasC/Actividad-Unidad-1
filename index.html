1. Qué es un Sistema Operativo

Es el software principal que gestiona el hardware de un computador y permite la interacción entre el usuario y el sistema.

2. Componentes de un Sistema Operativo

Núcleo (Kernel) Gestor de procesos Gestor de memoria Sistema de archivos Controladores de dispositivos (drivers) Interfaz de usuario (gráfica o por línea de comandos)

3. Tareas del Sistema Operativo

Administrar recursos del sistema Ejecutar y controlar procesos Gestionar memoria y almacenamiento Controlar periféricos Facilitar la interacción con el usuario

4. Funciones de un Sistema Operativo

Gestión de procesos Gestión de memoria Gestión de archivos Gestión de dispositivos Interfaz de usuario Seguridad y control de acceso

5. Características de un Sistema Operativo

Eficiencia Estabilidad Seguridad Interactividad Escalabilidad Multitarea y multiusuario

6. Tipos de Sistema Operativo

Monousuario y multiusuario Monotarea y multitarea Centralizado y distribuido En tiempo real De red Móviles

7. Sistemas Operativos Para PC

Windows macOS Linux (Ubuntu, Fedora, Debian)

8. Sistemas Operativos Móviles

Android iOS HarmonyOS KaiOS

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

Servidores: Linux, Windows Server Escritorio: Windows, macOS Móviles: Android, iOS Tiempo real: VxWorks, FreeRTOS

10. Compatibilidad de los Sistema Operativo

Depende del hardware, los drivers, el tipo de aplicaciones y el sistema de archivos. Algunos SO son compatibles con distintos tipos de arquitectura (x86, ARM).

11. Cuál es la función básica de un Sistema Operativo

Administrar el hardware y los recursos del sistema para permitir que el software funcione correctamente.

12. Mencione tres Sistema Operativo más utilizados

Windows macOS Linux

13. Menciona algunas tareas que realizan los Sistemas Operativos

Ejecutar aplicaciones Administrar memoria Controlar dispositivos de entrada/salida Gestionar archivos Programar tareas

14. Qué es un sistema operativo multitareas

Es un SO que puede ejecutar múltiples tareas o procesos al mismo tiempo.

15. Qué es un sistema operativo multiusuario

Es un SO que permite que varios usuarios trabajen con el sistema simultáneamente.

16. Cómo funciona un sistema operativo

Coordina el uso del hardware y software, asignando recursos y gestionando procesos para que las aplicaciones se ejecuten correctamente.

17. Cómo se utiliza un sistema operativo en un entorno gráfico

Mediante una interfaz gráfica (GUI) que permite al usuario interactuar con íconos, ventanas y menús con un ratón o pantalla táctil.

18. Cuándo inicia el funcionamiento del Sistema Operativo

Inicia al encender el computador, después de la ejecución del BIOS o UEFI (proceso de arranque).

19. Mencione las dos interfaces que forman parte del Sistema Operativo

Interfaz gráfica de usuario (GUI) Interfaz de línea de comandos (CLI)

20. Qué es un proceso en entorno de los sistemas operativos

Es una instancia de un programa en ejecución, que el sistema operativo gestiona con recursos asignados.

21. Qué es un sistema operativo distribuido y centralizado

Distribuido: el procesamiento está repartido entre varios equipos conectados en red. Centralizado: el procesamiento se realiza en un único equipo central.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

NTFS (Windows) FAT32 / exFAT (Windows) HFS+ / APFS (macOS) ext3 / ext4 / XFS / Btrfs (Linux)

23 .Mencione otras de las características actuales de los sistemas operativos

Soporte para virtualización Actualizaciones automáticas Seguridad avanzada (firewalls, cifrado) Compatibilidad con múltiples plataformas Conectividad en la nube
